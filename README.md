## Usage

### Download the template

```
$ boilr template download  <template-tag>
```

### Fork and use locally

```
$ git clone 
```

cd into it, then

```
$ boilr template save $(PWD) <template-tag>

# e.g. 
$ boilr template save $(PWD) boilr-makefile
```
